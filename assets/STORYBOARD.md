# Claude Cairn: launch video storyboard

**Format:** ~40s, 1920x1080, 60fps. No narration / no lyrics; a soft original
ambient bed (pads + gentle piano, generated by `assets/cairn_music.py`, royalty-free).
**Engine:** Manim CE 0.20.1 via `python3 -m manim`. Text / MarkupText only, no LaTeX.
**Look:** the Claude Code interface, dark window with traffic-light buttons, tool-call
dots (●), a rounded input box at the bottom that you TYPE into (the prompt then flies
up into the transcript and its tool output appears below it), **JetBrains Mono**, warm
**coral** accent. Real `assets/logo.png` on the intro and end card.

## Scenario (two unrelated threads, one chat)
1. **build** : implement a Transformer from scratch  ->  checkpoint `transformer`
2. **build** : implement a Soft Actor-Critic (SAC) agent  ->  checkpoint `sac`
Then two fresh sessions `/cairn:load` each and continue, context already there.
